A console is the combination of a monitor and keyboard. It is a rudimentary interface in which the monitor provides the output and the keyboard is used for input. ... Modern consoles often provide a graphical interface that can be used to control machines on a local network (LAN) or provide remote access to remote systems.

To Open Console Screen: First, Go on the website of which we need console screen. Then, Click Right-click Mouse Button. Choose INSPECT Option from the tab. After Elements tab, on the right-hand side There Will be console screen.

To print anything on console: Make A Script Tag in HTML Screen. Write console.log() and add the value or variable we want to print on console screen.
